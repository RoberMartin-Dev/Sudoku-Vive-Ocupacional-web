# Sudoku Vive Ocupacional

Aplicación web de Sudoku desarrollada para Vive Ocupacional.

## Características

- Generación procedural local.
- Solución única garantizada.
- Dificultades:
  - Fácil
  - Medio
  - Difícil
  - Avanzado
- Compatible con móviles y tablets.
- Modo competitivo con cronómetro.
- Descarga de puntuación en PNG.
- Sistema de impresión optimizado.
- Funciona completamente offline.
- No requiere backend.

## Tecnologías

- HTML5
- CSS3
- JavaScript Vanilla
